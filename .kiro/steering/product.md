# Product Overview

## System Name
Workflow System (工作流系统)

## Description
A simple authentication system built with Spring Boot (backend) and Vue 3 (frontend). The system provides user login functionality with JWT-style token-based authentication.

## Core Features
- User authentication via username/password
- BCrypt password hashing for security
- Token-based session management
- MySQL database persistence

## Target Users
- Internal system users
- Admin users with default credentials (admin/123456)

## Current State
- Basic login functionality implemented
- Single user model (User entity)
- RESTful API design
- Cross-origin requests enabled (development mode)
